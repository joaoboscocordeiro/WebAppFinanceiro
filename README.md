## Linguagem C#


   -	C# é uma linguagem de programação moderna, orientada a objetos e de tipagem forte, desenvolvida 
pela Microsoft como parte da plataforma .NET. É de código aberto, amplamente utilizada para desenvolver 
aplicativos web, desktop, móveis e jogos, destacando-se pela alta produtividade, segurança de tipo e 
uso de Garbage Collector para gerenciamento automático de memória.

----------------------------------------------------------------------------------------------------

## Dapper

   -	O Dapper é um Micro-ORM leve e de alto desempenho para .NET, ideal para C#, desenvolvido 
pelo Stack Overflow para mapear objetos de banco de dados rapidamente. Ele estende a IDbConnection 
com métodos de extensão como Query e Execute, permitindo consultas SQL diretas, mapeamento automático
de objetos, suporte a parâmetros e execução assíncrona, sendo mais rápido que ORMs completos.
